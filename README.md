## ゲームルール
ステージ上にある「エネルギーコア」を使い、ステージ上部の大砲を撃つためのエネルギーを集めることが目的のゲーム。エネルギーの集め方には「コアを持つ」「コアを投げて倒した相手から奪う」の2通りあり、状況に応じて使い分ける必要がある。最初に大砲を打ち出したプレイヤーが勝利となる。
